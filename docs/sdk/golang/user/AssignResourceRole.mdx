# AssignResourceRole

Assign a resource role to a specific user.

### Parameters

`ctx` -  The context of the request.<br/>
`userKey` - The key of the user to assign the role to.<br/>
`roleKey` - The key of the role to assign to the user.<br/>
`tenantKey` - The key of the user's tenant.<br/>
`resourceInstance` - The resource instance to assign the role with (e.g. "document:mydoc") .<br/>


### Implementation

```go
	resourceRoleAssignment, err := Permit.Api.Users.AssignResourceRole(ctx, "user-key", "role-key", "tenant-key", "document:doc1")
```
