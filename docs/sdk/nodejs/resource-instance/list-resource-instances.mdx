# list

List resource instances defined within the environment.

### Parameters

`tenant` (optional) - The tenant key or id to filter by<br/>
`resource` (optional) - The resource key or id to filter by<br/>
`page` (optional) - The page number of the results to fetch, starting at page 1<br/>
`perPage` (optional) - The number of results per page (maximum of 100)<br/>


### Implementation

```js
await permit.api.resourceInstances.list({
  tenant: "default",
  resource: "file",
  page: 1,
  perPage: 20,
});
```
