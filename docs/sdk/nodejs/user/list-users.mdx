# list

Lists users defined within the environment.

### Parameters

`search` (optional) - Search users by key, email, first name, or last name<br/>
`role` (optional) - Match users with a specific role<br/>
`page` (optional) - The page number of the results to fetch, starting at page 1<br/>
`perPage` (optional) - The number of results per page (maximum of 100)<br/>


### Implementation

```js
await permit.api.users.list({
  search: "@permit.io",
  role: "admin",
  page: 1,
  perPage: 20,
});
```
