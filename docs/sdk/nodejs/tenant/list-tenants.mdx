# list

Lists tenants defined within the environment.

### Parameters
`search` (optional) - Text search for the tenant name or key<br/>
`page` (optional) - The page number of the results to fetch, starting at 1<br/>
`perPage` (optional) - The number of results per page (maximum of 100)<br/>

### Implementation

```js
await permit.api.tenants.list({
  search: "example",
  page: 1,
  perPage: 20,
});
```
