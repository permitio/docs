# list

List resource instances defined within the environment.

### Parameters

`tenant` (optional) - The tenant key or id to filter by<br/>
`relation` (optional) - The relation id or key to filter by<br/>
`subject` (optional) - The subject to filter by, accepts either the resource instance id or resource_type:resource_instance<br/>
`subjectType` (optional) - The subject type to filter by, accepts resource type id or key<br/>
`object` (optional) - The object to filter by, accepts either the resource instance id or resource_type:resource_instance<br/>
`objectType` (optional) - The object type to filter by, accepts resource type id or key<br/>
`page` (optional) - The page number of the results to fetch, starting at page 1<br/>
`perPage` (optional) - The number of results per page (maximum of 100)<br/>


### Implementation

```js
await permit.api.relationshipTuples.list({
  tenant: "default",
  relation: "parent",
  subject: "file:file-1",
  subjectType: "file",
  object: "folder:folder-1",
  objectType: "folder",
  page: 1,
  perPage: 20,
});
```
