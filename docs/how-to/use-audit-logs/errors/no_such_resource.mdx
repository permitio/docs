---
title: Resource not found
---

This error occurred because the resource you provided in the permit check was not found. Ensure you input the correct, case-sensitive
resource name during the permit check.

## A solution to the issue

### Create resource with Permit web application

You can create resource with our [Policy management screen](https://app.permit.io/policy-editor)

### Create resource with the API

You can create resource with our [Resource management API](https://api.permit.io/v2/redoc#tag/Resources/operation/create_resource)

### Create resource with the SDK

Establish the resource with our [Python SDK](/sdk/python/usage-example#create-a-resource),
[Node SDK](/sdk/nodejs/resource/create-resource) and [other SDK's](/sdk/sdks-overview)
