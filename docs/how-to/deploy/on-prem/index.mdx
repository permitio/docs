## What You'll Deploy

The on-premises deployment includes:

- ✅ **Production-ready Kubernetes** (k3s) cluster
- ✅ **Complete Permit Platform** with all services
- ✅ **Air-gapped installation** - no internet required
- ✅ **Automatic security** configuration
- ✅ **High availability** support

## Deployment Process

1. **Prepare environment** - Git repository and server requirements
2. **Run installer** - Single command deployment
3. **Access platform** - Complete setup wizard
4. **Configure policies** - Deploy your authorization rules
